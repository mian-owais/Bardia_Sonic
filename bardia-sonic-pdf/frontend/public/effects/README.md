# Sound Effects Directory

This directory should contain subdirectories for various categories of sound effects:

1. Weather (`/weather/`)
   - thunder.mp3 (E1a)
   - rain.mp3 (E1b)
   - sunny_birds.mp3 (E1c)
   - night_frogs.mp3 (E1d)
   - gentle_breeze.mp3 (E1e)
   - blizzard.mp3 (E1f)

2. Miscellaneous (`/misc/`)
   - market.mp3 (E2a)
   - steps.mp3 (E2b)
   - knocking.mp3 (E2c)
   - glass_breaking.mp3 (E2d)
   - water.mp3 (E2e)
   - cup_filling.mp3 (E2f)
   - deep_breath.mp3 (E2g)
   - doorbell.mp3 (E2h)
   - phone.mp3 (E2i)
   - car_engine.mp3 (E2j)
   - crowd.mp3 (E2k)

3. Animal Sounds (`/animals/`)
   - birds.mp3 (E3a)
   - dog.mp3 (E3b)
   - horse.mp3 (E3c)
   - cat.mp3 (E3d)
   - owl.mp3 (E3e)

4. Beats (`/beats/`)
   - drumbeat.mp3 (E4a)
   - strings.mp3 (E4b)
   - dramatic.mp3 (E4c)
   - percussion.mp3 (E4d)

5. Machine Sounds (`/machines/`)
   - factory.mp3 (E5a)
   - spaceship.mp3 (E5b)
   - gunshot.mp3 (E5c)

For development, these can be placeholder files, but for production, they should be real audio files.
